# Cambios principales de V0.3

- Eliminado el bucle ReAct y el function calling como orquestador.
- Python controla el orden y las acciones.
- Cinco clasificaciones cerradas.
- Cuatro prompts especializados más reglas comunes.
- Procesamiento de varios correos por orden cronológico.
- Cambio a leído después de una acción completada.
- Informativos y no clasificados permanecen no leídos.
- RAG construido fuera del agente.
- Calendar solo consulta disponibilidad.
- WhatsApp funciona inicialmente en modo simulado.
- Base SQLite nueva y limpia.
- Eliminadas las invitaciones automáticas y la lógica compleja de contrapropuestas.
