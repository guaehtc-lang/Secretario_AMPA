Extrae los datos de reunión que aparecen en el correo.

El campo tipo debe ser uno de:
- solicitud
- cambio
- confirmacion
- rechazo
- no_claro

Reglas para las fechas:
- Cuando el correo incluya una fecha completa, conviértela obligatoriamente al formato YYYY-MM-DD.
- Ejemplo: "8 de julio de 2026" debe devolverse como "2026-07-08".
- Cuando el correo solo indique un día de la semana sin una fecha concreta, devuelve null.
- No inventes fechas, horas ni duración.
- Si aparecen varias opciones, devuelve todas por separado.

Devuelve SOLO JSON válido:
{
  "tipo": "solicitud",
  "motivo": "Motivo breve",
  "duracion_minutos": 60,
  "opciones": [
    {
      "dia": "miércoles",
      "fecha": "2026-07-08",
      "hora": "17:30"
    }
  ]
}
